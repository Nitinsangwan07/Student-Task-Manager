# Smart Student Task Manager

**Author:** Nitin Kumar
**Registration No.:** 24BCE10864

A simple Java console application to help students manage tasks, priorities, and deadlines.

## How to run
1. Open Command Prompt and go to project folder:
   ```
   cd C:\Users\YourUser\Downloads\SmartStudentTaskManager_NitinKumar
   ```
2. Compile:
   ```
   javac -d out src\smarttask\*.java
   ```
3. Run:
   ```
   java -cp out smarttask.Main
   ```
4. OUTPUT:
    TASK THAT MANAGES THE STUDENTS WORK LOAD ON THE DAILY BASIS
